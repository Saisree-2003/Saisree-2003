UAV Target Detection and Tracking

Bachelor's Thesis

Fusion of mmWave FMCW Radars and Vision-based Sensors

Overview

Developed a robust multi-sensor fusion system for autonomous drone applications, integrating mmWave FMCW radars with RGB and thermal cameras to enhance target detection accuracy and reliability in diverse environmental conditions.

Key Contributions

  • Led hardware installation and configuration of FMCW mmWave radars, RGB, and thermal cameras within ROS framework
  • Implemented synchronized data acquisition from multiple sensors for real-time processing
  • Designed and developed sensor fusion algorithms to combine radar and camera data
  • Achieved dynamic adaptability for improved detection in varying environmental scenarios

Technologies Used

ROS (Robot Operating System) Python OpenCV Sensor Fusion Computer Vision mmWave Radar

Project Details

Duration: 6 months

Status: Completed

Category: Computer Vision, Robotics

Achievements

  • Successful multi-sensor synchronization
  • Enhanced detection reliability
  • ROS-based implementation

Vessel Type Classification using Underwater Acoustic Data

Deep Learning Research

Lightweight CNN for Underwater Acoustic Signal Processing

Overview

Developed and implemented a novel lightweight convolutional neural network for precise classification of underwater vessels using acoustic data, achieving state-of-the-art performance while maintaining computational efficiency.

Key Contributions

  • Designed custom CNN architecture optimized for acoustic signal processing
  • Achieved 93.53% accuracy on VTUAD dataset
  • Optimized model size to 19.5 MB for deployment efficiency
  • Outperformed established CNN architectures in benchmark comparisons
  • Implemented robust preprocessing pipeline for underwater acoustic data
